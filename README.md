# zy27488-blog-replica

这是从 https://zy27488.github.io/ 公开静态站点恢复出的复刻版。

原纪念站仓库 zy27488/zy27488.github.io 不需要改动。这个仓库可以单独开启 GitHub Pages。

## 使用方式

1. 新建 GitHub 仓库，例如 zy27488-blog-replica。
2. 上传本目录全部文件。
3. 在仓库 Settings -> Pages 中选择 Deploy from a branch，分支 main，目录 /root。
4. 保存后等待 GitHub Pages 发布。

## 说明

这是 Hexo 生成后的静态文件复刻，不是原始 Hexo 源码。旧站显示使用 Hexo 6.2.0 和 Butterfly 主题。
